Web Development

Website for Ani95

Final project for Multimedia design and communication programme. During this project, together with my collegues we created interactive website for family-based sock factory in Bulgaria - Ani95. Our priority was to provide the users with great experience, design clear, understable UI as well as present and emphasize history, qualities and longevity of Ani95. That fully responsive website is made using ASP.NET Web forms, SQL database, Tailwind CSS, Animate CSS, fullPage.js and includes prototype of customization form for the customers.

Website for 1000FRYD

During this semester project together with my friends, we were able to create a prototype of a website for 1000FRYD. The main focus was to create a product that will clarify and present the profile of 1000FRYD and their organization. The challange was to create a calendar system, where admins can manage events and post them on the website. That was achived by using ASP.NET Web Forms and SQL Database. The website is responsive and uses also Skeleton CSS, Animate CSS and jQuery.
